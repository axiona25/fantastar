# ⚽ FANTASTAR

Fantacalcio Event-Based per la Serie A italiana.

## Quick Start

```bash
# Avvia i servizi
docker-compose up -d

# Verifica
curl http://localhost:8000/health
```

## Stack
- **Backend**: Python FastAPI
- **Database**: PostgreSQL
- **Cache**: Redis
- **Mobile**: Flutter
- **Web**: React
